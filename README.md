# Portfolio Website

A professional, high-performance portfolio website built with React and Anime.js. This project features a modular architecture, custom-built hooks for animations, and a responsive design optimized for all devices.

## Features

- **Modular Architecture**: Extracted components for header, experience, projects, skills, and education for maximum maintainability.
- **Custom Animation Hooks**: Sophisticated handling of typewriter effects and scroll-triggered animations.
- **Anime.js v4 Integration**: Utilizes the latest official ScrollObserver API for buttery-smooth, performance-optimized reveal effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **Interactive Elements**: Includes a physics-based skills playground and interactive project showcases.

## Technology Stack

- **React**: Core UI framework for a component-based UI.
- **Anime.js**: Industry-standard animation library for complex motion design.
- **Matter.js**: 2D physics engine for interactive components.
- **Vite**: Modern build tool for high-speed development and optimized production bundles.

## Development

To run the project locally, navigate to the `myCvPortfolio` directory and use the following commands:

```bash
npm install
npm run dev
```

## Deployment

The project is configured for automated deployment to GitHub Pages.

```bash
npm run deploy
```
